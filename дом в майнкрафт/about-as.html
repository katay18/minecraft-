День в МайнКрафт (Maincraft) сделать очень просто. Для этого потребуется следующий алгоритм действий: прописываем команду /time set day - эта команда автоматически установит время на день если нужен полдень, то команда будет выглядеть вот так: /time set 6000 (Делать это надо каждый раз, когда наступит ночь) если же нужна полночь то команда будет: time set 18000 если нужна глухая ночь: /time set 24 000 если нужны сумерки: /time set 18000
